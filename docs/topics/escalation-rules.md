## تصعيد فوري (خلال دقائق)
- اشتباه احتيال أو اختراق → فريق الأمن.
- تهديد رسمي بالتصعيد → فريق الاحتفاظ.
- انقطاع واسع النطاق → عمليات الشبكة.
- عميل VIP/مؤسسي → دعم متميز.
- قضايا قانونية/تنظيمية → الشؤون القانونية.

## تصعيد بعد المحاولة
| الحالة | القسم | شروط التصعيد |
|---|---|---|
| نزاع مالي كبير | Financial Disputes | بعد محاولة الحل |
| مشكلة تقنية معقدة | L2 Technical | بعد 3 محاولات فاشلة |
| طلب استثناء | Supervisor | خارج الصلاحيات |
| تكرار الاتصال | Case Manager | بعد أكثر من 3 مرات اتصال |

## مؤشرات تتطلب تصعيداً فورياً
- ذكر كلمات مثل: احتيال، سرقة، هيئة الاتصالات، محامي.
- مبلغ نزاع كبير (أكثر من 1000 ريال).
- استمرار المشكلة لأكثر من 7 أيام.

## بروتوكول التصعيد
1. اعتذار وتأكيد فهم المشكلة.
2. شرح سبب التصعيد للعميل.
3. نقل كامل التفاصيل دون تكرار.
4. إعطاء رقم مرجعي.
5. تحديد وقت انتظار متوقع.
