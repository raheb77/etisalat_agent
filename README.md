# CSR Decision Support Agent

Minimal decision-support backend with a lightweight CSR UI for querying `/query` and surfacing answers, confidence, category, citations, and handoff details.

## Quickstart
1. Activate your backend virtualenv.
2. Run `make dev`.
3. Open `http://localhost:5173`.

## Demo
1. Run `make dev`.
2. Open `http://localhost:5173` and submit a query.
3. Capture a screenshot and save it as `docs/demo.png` or record a GIF as `docs/demo.gif`.
4. Update the image below to match the asset you saved.
